# Components Guide

robo_appian groups Appian interactions into focused utility classes.

## Form Components
- `InputUtils`: Text fields, ids, and placeholders.
- `DateUtils`: Date inputs by label.
- `DropdownUtils`: Standard combobox dropdowns.
- `SearchDropdownUtils`: Searchable dropdowns.
- `SearchInputUtils`: Searchable input/listbox components.

## Navigation Components
- `ButtonUtils`: Buttons by visible text.
- `LinkUtils`: Links by exact text.
- `TabUtils`: Tab navigation and selection checks.
- `LabelUtils`: Visible text checks and clickable labels.

## Data Components
- `TableUtils`: Find tables, rows, and cell components by column name.

## Shared Helpers
- `ComponentUtils`: Waiting, clicking, XPath escaping, and convenience helpers.
- `BrowserUtils`: Multi-tab browser helpers.
- `RoboUtils`: Retry helpers.
